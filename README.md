# 🏨 StaySphere – PG & Hostel Management Platform

StaySphere is a full-stack PG & Hostel Management System that helps hostel owners manage tenants, rooms, rent, maintenance, visitors, expenses, attendance, agreements, and notices through a secure Admin Dashboard and Tenant Portal.

---

## 🚀 Features

### 🔐 Authentication
- JWT Authentication
- Admin & Tenant Login
- Role-Based Access Control

### 👨‍💼 Admin Portal
- Dashboard
- Tenant Management
- Room & Bed Management
- Rent Management
- Digital Agreements
- Maintenance Requests
- Visitor Management
- Expense Tracking
- Attendance & Check-In
- Notice Board

### 👤 Tenant Portal
- Tenant Dashboard
- View Rent Details
- View Digital Agreement
- Submit Maintenance Requests
- View Notices

## ⚙️ Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

Server runs at:

```
http://localhost:5000
```

### Frontend

Open the `frontend` folder using Live Server.
